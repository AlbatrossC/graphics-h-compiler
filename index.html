Why This Exists

graphics.h is a legacy graphics library still taught in Computer Graphics courses—especially in SPPU curriculum. But running it today requires jumping through hoops: installing Turbo C, configuring DOSBox, dealing with broken environments.

Students waste hours fighting setup instead of learning graphics programming.

This project eliminates that friction.

It provides two ways to run graphics.h programs instantly:

  • Online compiler – Works directly in your browser
  • VS Code extension – Integrates with your local development environment

No legacy tools. No manual configuration. Just write code and run it.

Built by an SPPU student who experienced this pain firsthand. The goal is simple: let students focus on learning graphics, not wrestling with 1990s software.
