About This Project

graphics.h is a decades-old graphics library that still survives in modern Computer Graphics courses — especially under the SPPU curriculum.

But running it today is painful.

Turbo C, DOSBox, broken installs, outdated environments — students are forced to fight the setup before they ever write code.

This project exists to remove that friction completely.

It lets you run graphics.h programs instantly:

  • • Online, directly in your browser
  • • Locally, using a modern VS Code extension

No legacy tools. No manual configuration.

Built by an SPPU student who faced the same frustration, this compiler is designed to help students focus on learning graphics programming — not wrestling with obsolete software.
